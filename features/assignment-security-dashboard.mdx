## Overview

The assignment security dashboard helps instructors identify and address potential security vulnerabilities, plagiarism, and suspicious patterns in student submissions.

## Accessing the dashboard

Navigate to **Manage Assignment** â†’ **Security** to view the security dashboard for any assignment.

## Security metrics

The dashboard displays:

### Submission patterns
- Unusual submission timing patterns
- Multiple submissions from the same IP address
- Rapid successive submissions

### Code similarity
- Potential plagiarism detection
- Similar code patterns across submissions
- Shared code segments

### Repository activity
- Commit history analysis
- Unusual repository access patterns
- External collaborator detection

## Taking action

When security issues are detected:

1. Review the flagged submissions
2. Compare similar submissions side-by-side
3. Contact students for clarification if needed
4. Apply academic integrity policies as appropriate

## Configuration

Configure security settings:
- Set similarity thresholds for plagiarism detection
- Enable or disable specific security checks
- Configure notification preferences for security alerts
