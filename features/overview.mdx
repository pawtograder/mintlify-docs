## Recent updates

This page provides an overview of the latest features and improvements added to Pawtograder.

## Assignment management

### Assignment dashboard
Get a comprehensive overview of assignment progress, submission status, and grading metrics all in one place.

[Learn more about the assignment dashboard →](/features/assignment-dashboard)

### Assignment security dashboard
Monitor and analyze security issues, detect potential plagiarism, and identify suspicious patterns in student submissions.

[Learn more about the security dashboard →](/features/assignment-security-dashboard)

### Test insights dashboard
Track student performance across individual test cases and identify common errors to improve your teaching.

[Learn more about test insights →](/features/test-insights-dashboard)

## Grading enhancements

### Deductive grading
Use deduction-only scoring mode for rubric criteria, perfect for style guides and penalty-based grading.

[Learn more about deductive grading →](/features/deductive-grading)

### Lab leader grading
Automatically assign reviews to lab section leaders for their students with bulk grading assignments.

[Learn more about lab leader grading →](/features/lab-leader-grading)

### Enhanced autograder rerun
Rerun autograder with version selection and submission reuse, making it easier to test and deploy grading changes.

[Learn more about autograder rerun →](/features/autograder-rerun)

## Student engagement

### Assignment leaderboard
Gamify learning with competitive leaderboards that display student rankings based on assignment scores.

[Learn more about leaderboards →](/features/assignment-leaderboard)

### Discussion karma system
Reward student engagement and helpful contributions with the karma system for discussion boards.

[Learn more about karma →](/features/discussion-karma)

### Error pins
Automatically suggest relevant discussion posts when students encounter common errors in their submissions.

[Learn more about error pins →](/features/error-pins)

## Assignment configuration

### Score capping
Set maximum scores for assignments to handle bonus points and extra credit while maintaining fair grading.

[Learn more about score capping →](/features/score-capping)

### Extended due date picker
Grant due date extensions with an intuitive date picker instead of calculating hours and minutes manually.

[Learn more about due date extensions →](/features/due-date-extensions)

## Additional improvements

### Grading annotations usability
Enhanced user interface for adding and managing grading annotations on student code.

### Export grades loading states
Added loading indicators to export grades buttons for better user feedback.

### Lab roster management
New lab roster management page for organizing and viewing lab section assignments.

### Discord integration
Enhanced Discord integration for discussion threads with deep links and channel linking.

## Getting started

To start using these features:
1. Update to the latest version of Pawtograder
2. Review the documentation for features you want to use
3. Configure settings in your course management dashboard
4. Communicate new features to students as appropriate

## Feedback

We're continuously improving Pawtograder based on instructor and student feedback. If you have suggestions or encounter issues with any of these features, please contact support.
