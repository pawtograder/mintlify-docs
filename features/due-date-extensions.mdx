## Overview

The extended due date picker allows instructors to grant due date extensions by selecting a specific date and time, rather than calculating hours and minutes manually.

## Using the due date picker

### Granting extensions

1. Navigate to **Manage Course** â†’ **Due Date Extensions**
2. Select the student(s) who need an extension
3. Choose the assignment
4. Use the date picker to select the new due date
5. Click **Grant Extension**

### Extension details
- Select both date and time
- View the extension duration automatically calculated
- See the original due date for reference
- Add notes explaining the extension reason

## Extension types

### Individual extensions
Grant extensions to specific students for specific assignments.

### Bulk extensions
Grant the same extension to multiple students at once:
1. Select multiple students
2. Choose the assignment
3. Set the extended due date
4. Apply to all selected students

### Lab section extensions
Grant extensions to all students in a specific lab section.

## Managing extensions

### View active extensions
- See all active extensions in one place
- Filter by assignment or student
- Sort by extension date
- Export extension data

### Modify extensions
- Update extension dates as needed
- Remove extensions if no longer needed
- Add notes or comments

### Extension notifications
Students are automatically notified when:
- An extension is granted
- An extension is modified
- An extension is removed

## Best practices

- Document extension reasons for record-keeping
- Be consistent with extension policies
- Communicate extension policies clearly to students
- Review extensions regularly to ensure fairness
