## Welcome to Pawtograder

Pawtograder is a comprehensive autograding and course management platform designed for computer science education. It provides automated grading, peer review management, discussion boards, and analytics to support large-scale programming courses.

## Core features

### Autograding

- Automated code evaluation with GitHub integration
- Customizable test suites and rubrics
- Lint checking and style enforcement
- Execution time tracking
- Multiple submission attempts

### Peer review

- Configurable review assignments
- Rubric-based grading
- Anonymous grading options
- Review assignment automation
- Grading conflict management

### Discussion boards

- Threaded discussions with topics
- Instructor-only posts
- Real-time updates
- Markdown support
- File attachments

### Analytics

- Student performance tracking
- Test difficulty analysis
- Engagement metrics
- Gradebook integration
- CSV exports

## Platform architecture

Pawtograder is built with:

- **Frontend**: Next.js with React
- **Backend**: Supabase (PostgreSQL + Edge Functions)
- **Authentication**: Supabase Auth with magic links
- **Real-time**: Supabase Realtime for live updates
- **Storage**: GitHub for code repositories
- **Deployment**: Vercel for hosting
