## May 2026

### May 20, 2026

#### Bug fixes
- **Code renderer follows app theme** - The code renderer in submission views now follows the app's light/dark mode setting instead of the OS-level preference ([#711](https://github.com/pawtograder/platform/pull/711))
- **Submission history reflects real-time review changes** - The submission history panel now updates in real time when review changes are made ([#598](https://github.com/pawtograder/platform/pull/598))

### May 19, 2026

#### Instructor features
- **Commit history and manual grading trigger** - Instructors can now view the full commit history for any student repository and manually trigger grading for any commit, bypassing due-date deadlines and submission limits. A **Commits** column appears in the assignment management table, and a **Commit History** button is available in the submission view ([#764](https://github.com/pawtograder/platform/pull/764))
- **Instructor page performance** - The instructor assignment management page now uses streaming and virtualization for significantly faster load times on large courses ([#708](https://github.com/pawtograder/platform/pull/708))
- **Sync handout to repository for large files** - Syncing the handout to student repositories now works correctly for files larger than 5 MB ([#749](https://github.com/pawtograder/platform/pull/749))
- **Async archive queue for large assignment deletes** - Deleting assignments with many repositories now uses an async queue to avoid timeouts ([#765](https://github.com/pawtograder/platform/pull/765))

#### Bug fixes
- **Gradebook recalculation on active submission changes** - The gradebook now correctly recalculates when the active submission changes ([#763](https://github.com/pawtograder/platform/pull/763))

### May 18, 2026

#### Discussion improvements
- **Mark thread as duplicate** - Graders and instructors can now merge a duplicate discussion thread into another thread. The merged post shows a banner with the original subject and who merged it, and the author receives a notification ([#727](https://github.com/pawtograder/platform/pull/727))

#### Instructor features
- **GitHub account diagnostics** - Instructors can now diagnose and fix GitHub account issues for individual students from the Enrollments page. The diagnostics modal shows linked account, username changes, org membership, team membership, and database org status, with actions to sync permissions or unlink the GitHub identity ([#750](https://github.com/pawtograder/platform/pull/750))
- **Course-wide regrade requests dashboard** - A new **All regrade requests** page (accessible from Manage → Assignments) shows every regrade request across all assignments in the course, with filtering by status, assignment, rubric check, student, assignee, and escalation status ([#733](https://github.com/pawtograder/platform/pull/733))
- **Assessment export CLI** - The CLI now supports `assessment export` and `assessment deanonymize` commands for streaming privacy-controlled snapshots of rubrics, scores, autograder results, hints, and gradebook data ([#746](https://github.com/pawtograder/platform/pull/746))
- **Cleanup individual repositories** - A new **Cleanup individual repositories** button on the assignment Repositories page removes mistaken individual repos when an assignment was converted to group-only after release ([#720](https://github.com/pawtograder/platform/pull/720))

#### Accessibility and usability
- **Comprehensive accessibility fixes** - Multiple accessibility and usability improvements across the platform ([#753](https://github.com/pawtograder/platform/pull/753))

## April 2026

### April 29, 2026

#### Gradebook enhancements
- **Improved Expression Builder editor** - The full-screen Expression Builder now uses a Monaco-based editor with syntax highlighting, autocomplete for built-in functions and column slugs, and hover tooltips showing intermediate values for any sub-expression. Per-line result annotations are displayed inline next to each statement ([#735](https://github.com/pawtograder/platform/pull/735))
- **Gradebook table select-all/none filters** - Column and section filter dropdowns in the gradebook table now include **Select all** and **Select none** toolbar buttons for faster multi-select filtering ([#735](https://github.com/pawtograder/platform/pull/735))

#### Autograding fixes
- **Fail loudly when no files match submissionFiles patterns** - If no repository files match the configured `submissionFiles` glob patterns, the submission is now immediately rejected with a descriptive 400 error listing the unmatched patterns and guiding instructors to check `pawtograder.yml` ([#739](https://github.com/pawtograder/platform/pull/739))

### April 24, 2026

#### Grading improvements
- **Clear TA messaging when grading feedback is released** - When a TA attempts to add or change marks on a released submission review, Pawtograder now shows a clear warning explaining the restriction and directing them to ask an instructor to unrelease the review ([#724](https://github.com/pawtograder/platform/pull/724))
- **Group mentor column in instructor assignment table** - The instructor submissions table now shows a **Group mentor** column (with multi-select filter) when the assignment has groups with assigned mentors ([#731](https://github.com/pawtograder/platform/pull/731))

#### Bug fixes
- **Fixed individual score display on group assignments** - Student views now correctly show individual scores on group assignments ([#729](https://github.com/pawtograder/platform/pull/729))
