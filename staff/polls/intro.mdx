# Polls Overview

Pawtograder's Polls feature provides a quick and easy way for instructors to conduct live in-class polling. Polls are designed for real-time engagement during lectures, allowing instructors to gauge student understanding, gather quick feedback, or facilitate interactive discussions.

## Key Features

- **Quick Creation**: Create polls in seconds with a simple visual builder
- **Real-Time Results**: See student responses update instantly as they come in
- **Multiple Poll Types**: Support for single choice and multiple choice questions
- **Live Visualizations**: Display results to the class with built-in bar charts
- **Presentation Mode**: Full-screen display for classroom projection with embedded QR code
- **Automatic Closing**: Polls automatically close after one hour to ensure timely responses
- **Anonymous or Authenticated**: Choose whether students must log in to respond

## Use Cases

Polls are ideal for:

- **Comprehension Checks**: Quickly assess if students understand a concept before moving on
- **Discussion Starters**: Gather opinions to spark class discussions
- **Attendance**: Quick polls at the start of class (use "Require Login" for tracking)
- **Feedback**: Get instant feedback on pace or clarity of instruction
- **Concept Voting**: Let students vote on topics to cover or questions to answer

## Polls vs Surveys

| Feature | Polls | Surveys |
|---------|-------|---------|
| Purpose | Quick in-class engagement | Detailed feedback collection |
| Duration | Auto-closes after 1 hour | Configurable due dates |
| Question Count | Single question per poll | Multiple questions |
| Response Editing | One response per poll | Editable until submitted |
| Real-Time Results | Live updates during class | Aggregated after collection |
| Best For | Lectures, quick checks | Course evaluations, detailed feedback |

## Getting Started

To use Polls, navigate to your course and select "Polls" from the course navigation. From there, you can:

1. **Create new polls** using the visual builder
2. **Manage existing polls** (open, close, delete)
3. **View responses** with real-time analytics
4. **Present results** in fullscreen mode for the classroom

<Note>
Polls are scoped to your class - you can only view and manage polls for courses you are an instructor or grader for.
</Note>

## Quick Start

1. Go to **Polls** in your course navigation
2. Click **Create Poll**
3. Click **Open Visual Builder** to design your question
4. Choose whether to **Require login**
5. Click **Publish Poll** to make it live
6. Share the QR code or link with students
7. Watch responses come in real-time!
