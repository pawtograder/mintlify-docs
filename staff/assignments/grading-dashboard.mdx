# Grading Status Dashboard

The grading status dashboard provides instructors with a comprehensive overview of grading progress across all assignments. This helps you track which assignments need attention and monitor grader workload.

## Overview

The dashboard displays:

- **Assignment-level progress**: See completion rates for each assignment
- **Grader-level progress**: Track individual grader performance
- **Pending tasks**: Identify submissions waiting for review
- **Completion trends**: Monitor grading velocity over time

## Accessing the Dashboard

Navigate to the grading status dashboard from the course management interface. The dashboard is available to instructors and head TAs.

## Assignment Progress

For each assignment, you can see:

- **Total submissions**: Number of student submissions received
- **Graded submissions**: How many have been fully graded
- **In-progress submissions**: Currently being reviewed by graders
- **Unassigned submissions**: Waiting for grader assignment
- **Completion percentage**: Overall grading progress

### Color-Coded Status

The dashboard uses color coding to highlight priority:

- **Green**: Grading is complete or on track
- **Yellow**: Some submissions pending review
- **Red**: Significant backlog or approaching deadline

## Grader Performance

Track individual grader progress:

- **Assigned tasks**: Number of submissions assigned to each grader
- **Completed tasks**: How many they've finished
- **Average time**: Time spent per submission
- **Pending reviews**: Outstanding grading tasks

This helps you:

- Identify graders who may need support
- Balance workload across the team
- Recognize high-performing graders
- Spot bottlenecks in the grading process

## Filtering and Sorting

Use filters to focus on specific data:

- **By assignment**: View progress for individual assignments
- **By grader**: See all tasks for a specific grader
- **By status**: Filter by pending, in-progress, or completed
- **By due date**: Prioritize assignments by deadline

The grading status dashboard provides a comprehensive view of grading progress across all assignments in your course. Use this dashboard to track which submissions have been graded, identify bottlenecks, and ensure timely feedback to students.

## Accessing the Dashboard

Navigate to the Assignments section and select "Grading Status" to view the dashboard.

## Dashboard Features

### Assignment Overview

The dashboard displays all assignments with key metrics:

- **Total submissions**: Number of student submissions received
- **Graded submissions**: Count of completed grading reviews
- **Pending submissions**: Submissions awaiting grading
- **Completion percentage**: Visual progress indicator

### Grading Progress Tracking

For each assignment, you can see:

- **Grader assignments**: Which staff members are assigned to grade
- **Due dates**: Grading task deadlines
- **Status indicators**: Visual cues for on-track, at-risk, or overdue grading

### Filtering and Sorting

Use filters to focus on specific assignments:

- Filter by assignment status (active, past due, upcoming)
- Sort by grading completion percentage
- View only assignments with pending grading tasks

## Taking Action

From the dashboard, you can:

- **Reassign tasks**: Move submissions between graders
- **Send reminders**: Notify graders about pending work
- **Bulk assign**: Distribute unassigned submissions
- **Release grades**: Publish completed reviews to students

## Best Practices

### Regular Monitoring

- Check the dashboard daily during grading periods
- Set up a grading schedule and track adherence
- Address bottlenecks promptly

### Workload Balance

- Ensure even distribution of grading tasks
- Consider grader experience when assigning complex submissions
- Provide additional support to graders falling behind

### Communication

- Use the dashboard data to inform team meetings
- Share progress updates with graders
- Celebrate milestones and completion

### Quality Assurance

- Monitor average grading time for consistency
- Review completed work from new graders
- Use the dashboard to identify outliers that may need review
