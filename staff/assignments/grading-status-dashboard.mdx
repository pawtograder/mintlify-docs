# Grading Status Dashboard

The grading status dashboard provides instructors with a comprehensive view of grading progress across all assignments in the course. This dashboard helps you track which submissions have been graded, identify bottlenecks, and ensure timely completion of grading tasks.

## Accessing the Dashboard

Navigate to the grading status dashboard from the course navigation menu to view real-time grading progress across all assignments.

## Dashboard Features

### Assignment Overview

The dashboard displays key metrics for each assignment:

- **Total submissions**: Number of student submissions received
- **Graded submissions**: Count of completed grading reviews
- **Pending submissions**: Submissions awaiting grading
- **Completion percentage**: Visual progress indicator

### Grading Progress Tracking

Monitor the status of grading assignments:

- View which graders have completed their assigned tasks
- Identify overdue grading assignments
- Track grading velocity and estimated completion times
- See which submissions are in progress vs. not started

### Filtering and Sorting

Customize your view to focus on specific information:

- Filter by assignment, grader, or completion status
- Sort by due date, submission count, or completion percentage
- Search for specific students or grading tasks

## Using the Dashboard

The grading status dashboard helps you:

- **Identify bottlenecks**: Quickly spot assignments with low completion rates
- **Balance workload**: Reassign tasks if certain graders are falling behind
- **Meet deadlines**: Ensure grading is completed before grade release dates
- **Track accountability**: Monitor individual grader performance and completion rates

## Integration with Grading Assignments

The dashboard integrates seamlessly with the [grading assignments](/staff/assignments/grading-assignments) feature, allowing you to:

- Click through to view specific grading tasks
- Reassign incomplete tasks to other graders
- Send reminders to graders with pending assignments
- Export grading progress reports
