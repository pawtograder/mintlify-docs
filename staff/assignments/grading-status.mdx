# Grading Status Dashboard

The grading status dashboard provides a comprehensive view of grading progress for an assignment, helping you track which submissions have been graded and manage grader assignments.

## Accessing the Dashboard

Navigate to an assignment's management page and view the grading status overview on the main assignment page.

## Dashboard Overview

The grading status dashboard displays:

- **Total submissions**: Number of student submissions for the assignment
- **Graded submissions**: How many have been fully graded
- **Pending submissions**: Submissions awaiting grading
- **Grader assignments**: Which graders are assigned to which submissions

## Grading Progress Tracking

### Overall Progress

The dashboard shows:
- **Completion percentage**: Visual progress bar showing grading completion
- **Submissions by status**: Breakdown of graded vs. pending submissions
- **Average grading time**: How long graders are taking per submission

### Per-Grader Statistics

View grading progress for each grader:
- **Assigned submissions**: How many submissions each grader has
- **Completed submissions**: How many they've finished grading
- **Pending submissions**: What's still in their queue
- **Average time per submission**: Grading efficiency metrics

## Managing Grader Assignments

### Assigning Graders

You can assign graders to submissions in two ways:

1. **Bulk assignment**: Assign multiple submissions to a grader at once
2. **Individual assignment**: Assign specific submissions to specific graders

### Reassigning Submissions

If a grader is overloaded or unavailable:
1. Select the submissions to reassign
2. Choose a new grader from the dropdown
3. Click **Reassign** to update the assignments

### Partial Rubric Grading

Instructors can assign graders to specific parts of the rubric rather than entire submissions:

- **Full rubric assignment**: Grader handles all rubric items for a submission
- **Partial assignment**: Grader only grades specific rubric parts

This allows you to distribute grading work more efficiently and leverage grader expertise.

### For Graders

When you have submissions assigned to grade:

2. **Grade submissions**: Click on a submission to open the grading interface
3. **Track your progress**: See how many submissions you have left to grade

## Grading Metrics

The dashboard provides insights to help manage the grading process:

### Time Estimates

- **Average time per submission**: Based on completed gradings
- **Estimated time remaining**: Projected time to complete all pending submissions
- **Grader efficiency**: Compare grading times across graders

### Quality Metrics

- **Score distribution**: How grades are distributed across submissions
- **Rubric item usage**: Which rubric items are most commonly applied
- **Feedback length**: Average length of grader comments

## Best Practices

### Balanced Assignment

- Distribute submissions evenly among graders
- Consider grader expertise when assigning specific rubric parts
- Monitor workload and reassign if needed

### Timely Grading

- Set internal deadlines for graders
- Monitor progress regularly
- Identify and address bottlenecks early

### Quality Assurance

- Review a sample of each grader's work
- Ensure consistent application of rubric criteria
- Provide feedback to graders on their grading quality

### Communication

- Keep graders informed of deadlines
- Share common issues or clarifications
- Recognize graders who complete work efficiently

## Grading Notifications

Graders receive notifications when:
- New submissions are assigned to them
- Submission deadlines are approaching
- Students submit regrade requests

Configure notification preferences in your user settings.

The grading status dashboard provides instructors with a comprehensive view of grading progress for an assignment. This dashboard helps you track which submissions have been graded, identify bottlenecks, and ensure timely completion of grading tasks.

## Accessing the Dashboard

Navigate to an assignment and select the **Grading Status** tab to view the dashboard. The dashboard is available to instructors and head TAs.

## Dashboard Features

The grading status dashboard displays:

- **Overall progress**: Percentage of submissions that have been graded
- **Grading breakdown**: Number of submissions in each state (not started, in progress, completed)
- **Grader assignments**: Which staff members are assigned to which submissions
- **Completion timeline**: When grading tasks were completed
- **Outstanding tasks**: Submissions that still need grading

## Monitoring Grading Progress

Use the dashboard to:

1. **Track completion rates**: See how many submissions have been fully graded
2. **Identify delays**: Find submissions that are taking longer than expected
3. **Balance workload**: Ensure grading tasks are distributed evenly among staff
4. **Plan releases**: Determine when you can release grades to students

## Grading States

Submissions can be in one of several states:

- **Not started**: No grading has been completed
- **In progress**: Some rubric items have been graded, but the review is not complete
- **Completed**: All required rubric items have been graded (but not necessarily released)
- **Released**: Grades have been released to students

<Note>
Completing a review does not automatically release it to students. Instructors must explicitly release grades when ready.
</Note>

## Using the Dashboard for Grading Assignments

The grading status dashboard integrates with the grading assignment system. When you assign grading tasks to staff:

- The dashboard shows which grader is assigned to each submission
- You can track whether assigned graders have completed their tasks
- Overdue grading tasks are highlighted
- You can reassign tasks directly from the dashboard if needed

<Tip>
Check the grading status dashboard regularly during the grading period to ensure all submissions are being processed on time.
</Tip>
