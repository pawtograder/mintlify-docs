# Grading Status Dashboard

The grading status dashboard provides a comprehensive overview of grading progress across all assignments in your course, helping you track which assignments need attention and monitor grader workload.

## Accessing the Dashboard

Navigate to the **Grading Status** section from the course navigation menu to view the dashboard.

## Dashboard Overview

The dashboard displays key metrics for each assignment:

- **Assignment name** and due date
- **Total submissions** received
- **Grading progress** (percentage complete)
- **Ungraded submissions** count
- **Assigned graders** and their progress
- **Average grading time** per submission

## Monitoring Grading Progress

For each assignment, you can see:

### Overall Progress
- Visual progress bar showing percentage of submissions graded
- Count of completed vs. total submissions
- Estimated time to completion based on current grading pace

### Grader Breakdown
- List of assigned graders
- Number of submissions assigned to each grader
- Completion status for each grader
- Overdue assignments highlighted in red

### Submission Status
- Submissions awaiting grading
- Submissions in progress
- Completed submissions
- Submissions with grading conflicts

## Taking Action

From the grading status dashboard, you can:

- **Reassign tasks**: Click on a grader to view their assignments and reassign if needed
- **Send reminders**: Notify graders about pending assignments approaching their due dates
- **View submissions**: Click on any assignment to jump directly to the grading interface
- **Adjust deadlines**: Extend grading deadlines for assignments that are behind schedule

## Filtering and Sorting

Use the filters at the top of the dashboard to:

- Show only assignments with pending grading tasks
- Filter by assignment due date or grading deadline
- Sort by grading progress (least to most complete)
- Filter by specific graders or sections

## Best Practices

- **Check regularly**: Review the dashboard weekly to ensure grading stays on track
- **Balance workload**: Use the grader breakdown to identify if some graders are overloaded
- **Set realistic deadlines**: Allow sufficient time between assignment due dates and grade release dates
- **Address bottlenecks early**: Reassign tasks or recruit additional graders when progress falls behind
