# Repository Analytics

Pawtograder provides analytics for student assignment repositories, allowing you to track commit activity, repository engagement, and development patterns across your course.

## Viewing Repository Analytics

Navigate to an assignment's analytics page to see repository statistics for all student submissions.

### Available Metrics

The repository analytics dashboard displays:

- **Commit count**: Total number of commits made to each student's repository
- **Commit timeline**: Visual representation of when commits were made
- **Active contributors**: Number of unique contributors to group repositories
- **Repository activity**: Last commit timestamp and activity patterns
- **Commit frequency**: Distribution of commits over time

## Understanding Commit Patterns

### Individual Assignments

For individual assignments, the analytics show:

- Single student's commit history
- Commit frequency over the assignment period
- Timestamps of first and last commits
- Total commit count

### Group Assignments

For group assignments, the analytics include:

- Combined commit history from all group members
- Individual contribution breakdown by student
- Commit distribution among group members
- Group collaboration patterns

## Using Analytics for Assessment

Repository analytics can help you:

- **Identify early starters vs. procrastinators**: See when students begin working on assignments
- **Detect collaboration issues**: In group projects, identify uneven contribution patterns
- **Monitor progress**: Track whether students are making steady progress or cramming before deadlines
- **Verify academic integrity**: Unusual commit patterns may warrant further investigation
- **Provide targeted support**: Reach out to students with low or no commit activity

<Warning>
Commit count alone is not a reliable measure of effort or quality. Some students make many small commits while others make fewer, larger commits. Use analytics as one data point among many when assessing student work.
</Warning>

## Exporting Analytics Data

Click **Export to CSV** to download repository analytics for external analysis or record-keeping.

The CSV export includes:

- Student name and email
- Repository URL
- Total commit count
- First commit timestamp
- Last commit timestamp
- Commit frequency metrics

<Tip>
Combine repository analytics with gradebook data and submission timestamps to get a complete picture of student engagement and progress.
</Tip>

## Privacy and Access

- Only instructors and graders can view repository analytics
- Students cannot see analytics for other students' repositories
- Analytics data is derived from GitHub repository metadata
- No code content is analyzed or stored, only commit metadata
